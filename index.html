O Projeto

Olá, esse site foi elaborado com o intuito de ser um treinamento para desenvolver minhas habilidades criativas e logicas em desenvolvimento front-end, para a criação da página foram utilizadas as tecnologias: Html, Css, Sass e Javascript. No momento o site se encontra em sua versão inicial, porém está em constante evolução para alcançar o objetivo de ser um site referência para solução de problemas complexos de matematica.
